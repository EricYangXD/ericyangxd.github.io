### 睡眠排序

-   睡眠排序就是构造 n 个线程，让线程和排序的 n 个数对应。

-   例如对于[4,2,3,5,9]这样一组数字，就创建 5 个线程，每个线程睡眠 4s，2s，3s，5s，9s。这些线程睡醒之后，就把自己对应的数报出来即可。这样等所有线程都醒来，排序就结束了。

-   但睡眠排序由于多线程的问题，在真正实现上也有困难。

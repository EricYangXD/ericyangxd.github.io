    # 如何录入通达信公式

    导入技术指标公式的步骤:

    1. 以 Windows 版本为准，Mac 版根本不能用，同花顺也一样。
    2. 打开通达信软件；
    3. 点击功能——公式系统——公式管理器；
    4. 选择技术指标公式——新建；
    5. 填写名称，复制公式粘贴到大框框里，如果公式前面有参数的，把参数填写到大框框上面的参数表中，选择公式类型为其它类型，选择画线方法为主图叠加或者副图；
    6. 测试公式——确定。
